# projX-la-spmv

## 项目名称

基于龙芯3A5000设计实现一个高效的SPMV（稀疏矩阵乘向量）

## 支持单位

龙芯中科技术股份有限公司

## 项目描述

针对LoongArch架构，使用多线程，矩阵分块和SIMD等技术，优化实现CSR/CSC格式SPMV（稀疏矩阵乘向量）。具体内容包括：

1） 调研已有的SPMV算法优化技术。
2） 实现基于龙芯3A5000的算法优化，进行测试和调优，生成相应文档。
